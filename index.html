Privacy Policy

Effective date: January 2026

Simple PDF Joiner ("the app") respects your privacy. This application is designed to work completely offline and does not collect, store, or transmit any personal data.

Information Collection

The app does not collect any personal information, including but not limited to:

  • Names
  • Email addresses
  • Phone numbers
  • Location data
  • Device identifiers

File Access

The app accesses PDF files only when the user explicitly selects them using the Android system file picker. All file processing happens locally on the user's device. No files are uploaded, shared, or transmitted outside the device.

Internet Usage

Simple PDF Joiner does not require an internet connection and does not communicate with any external servers.

Third-Party Services

This app does not use third-party analytics, advertising SDKs, or tracking services.

Data Security

Since no personal data is collected or stored, there is no risk of data leakage. All merged PDF files remain on the user's device.

Children’s Privacy

Simple PDF Joiner does not knowingly collect any personal information from children.

Changes to This Policy

If this privacy policy changes in the future, updates will be posted on this page.
