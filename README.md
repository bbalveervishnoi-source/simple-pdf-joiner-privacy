# Privacy Policy – Simple PDF Joiner

Last updated: January 2026

Simple PDF Joiner respects your privacy. This application is designed to work fully offline and does not collect or transmit any personal data.

## Information Collection and Use

Simple PDF Joiner does NOT collect, store, share, or transmit any personal or sensitive user data.

- No user accounts
- No analytics
- No advertisements
- No internet access
- No background data collection

## File Access

The app uses Android’s system file picker (Storage Access Framework) solely to allow users to select PDF files for merging.  
All file processing occurs locally on the user’s device.

Selected files are never uploaded, shared, or stored outside the device.

## Data Sharing

No data is shared with any third party.

## Children’s Privacy

Simple PDF Joiner does not knowingly collect any personal information from children.

## Security

Because the app works entirely offline, user data remains fully under the user’s control at all times.

## Changes to This Policy

If the app’s behavior changes in the future, this privacy policy will be updated accordingly.
